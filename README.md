# Ficha-de-Artigo
Versões Desktop e APK que pegam dados de uma api e imprime em uma Etiquetadora Elgin.

As configurações necessárias para que o programa funcionasse corretamente foram:
Colocar a impressora na rede, 
Configurar um IP nela, 
Liberar comunicação entre o dispositivo que está no wifi para a empressora que está cabeada no firewall.

Foi utilizado React, Python, NodeJS, TailwindCSS, Javascript
